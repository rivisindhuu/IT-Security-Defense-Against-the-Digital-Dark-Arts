# Windows Basic System Security Check

This project demonstrates a basic system security check on a Windows system, including password strength, firewall status, and software updates. Screenshots are included for each step.

---

## Step 1: Check Password Strength

On Settings

1. Went to Accounts → Sign-in options.
then checked the password complexity.

<img width="754" height="334" alt="password on local security policy" src="https://github.com/user-attachments/assets/189f4992-686a-4af6-b923-9e5893d70b06" />


### **Improvements / Suggestions:**
- Use strong passwords (12+ characters with symbols/numbers)
- Enable MFA
- Avoid personal info in passwords

---

## Step 2: Check Firewall Status


1.From the  windows search ,I opened Windows Security.
 
2.Ensured that the firewall is On for all network types on the Firewall & network protection.
   <img width="606" height="452" alt="Domain, Private, Public" src="https://github.com/user-attachments/assets/ee7f2574-a51e-4b5e-945e-3738788d3ce0" />

3.Review allowed apps in the firewall settings.


<img width="696" height="530" alt="Check Firewall Status" src="https://github.com/user-attachments/assets/1def0e7a-36c3-41cb-9193-719b03ae6bd7" />

### **Improvements / Suggestions:**
- Keep firewall enabled at all times
- Allow only trusted applications
- Review firewall logs regularly

2. Go to **Virus & threat protection**.
3. Click **Quick scan** (or Full scan for thorough scanning).

## **Conclusion:**  
The system security check confirms that the firewall is active, passwords are strong, and software is not up to date. Enabling encryption and maintaining regular updates further improve security.

| Security Area           | Status Checked | Improvements / Suggestions |
|-------------------------|----------------|----------------------------|
| Password Strength       |  Weak | Use strong passwords, enable MFA |
| Firewall                |  On    | Keep firewall enabled, allow only trusted apps |
| Software Updates        |  Pending | Enable automatic updates, remove old software |
| Antivirus Scan          |  Clean | Run regular scans, update definitions |
